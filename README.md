# Python Docs Pessoais

Repositório com minha "documentação" de Python: comandos, conceitos e exemplos curtos.
A ideia é registrar tudo que eu aprender de forma organizada e fácil de consultar.

## Estrutura
- `docs/` → páginas em Markdown com explicações e exemplos
- `examples/` → scripts pequenos de demonstração

## Como contribuir (>eu mesmo<)
1. Aprendeu algo novo? Crie/edite uma página em `docs/`
2. Adicione exemplos curtos e “pegadinhas” (erros comuns)
3. Commit + push
