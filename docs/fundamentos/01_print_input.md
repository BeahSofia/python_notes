# 01 - print, input, variáveis e tipos

## print()
Serve para mostrar algo no terminal.
```python
print("Olá, mundo!")
print(10)
print(10+5)
```

# Variáveis
Guarda valores para usar depois.
```python
nome = "Beatriz"
idade = 20
print(nome, idade)
```

# f-string
Forma prática de montar textos com variáveis.
```python
nome = "Bea"
print(f"Olá, {nome}!")
```

# input()
Lê texto digitado pelo usuário (vê como str).
```python
nome = input("Qual o seu nome?")
print(f"Prazer,{nome}!")
```

# Conversão de tipos
Para fazer contas, você normalmente converte:
```python
idade = int(input("Idade: "))
print(idade + 1)
```
